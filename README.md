# Virtual_Reality
Bowling & Firing Range VR Prototypes

Bowling Scene Features:
-Locomotion
-Grabbable Bowling Balls
-Knockable Bowling Pins
-UI World Space Score
-UI World Space Reset Button

Firing Range Scene Features:
-Locomotion + Animated Hands
-Grabbable Pistol + Shooting Mechanics + Firing Animations + Ammo UI
-Grabbable Magazines + Attachable to Pistol
-Spawning + Moving Targets
-UI World Space Score
-Magazine Spawner Pushable Button
